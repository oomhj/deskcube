# 8266Toy

ESP8266 NodeMCU + ST7789V 240×240 TFT

## 接线

| # | 屏幕引脚 | NodeMCU | GPIO | 说明 |
|---|---------|---------|------|------|
| 1 | GND | GND | — | 地 |
| 2 | DC | D3 | GPIO0 | 数据/命令 |
| 3 | CS | GND | — | 片选（直接接地） |
| 4 | SCK | D5 | GPIO14 | SPI 时钟 |
| 5 | SDA | D7 | GPIO13 | SPI 数据 (MOSI) |
| 6 | REST | D4 | GPIO2 | 复位 |
| 7 | VDD | 3.3V | — | 电源 |
| 8 | GND | GND | — | 地 |
| 9 | LEDA | NPN 集电极 | GPIO5 | 背光阳极 (NPN 控制) |
| 10 | LEDK | GND | — | 背光阴极 |

**背光**：GPIO5 **LOW = 亮**，HIGH = 灭

## TFT_eSPI 配置（`User_Setup.h`）

配置文件为项目根目录的 `User_Setup.h`，通过编译参数 `-include User_Setup.h` 加载。

```cpp
#define USER_SETUP_LOADED            // 让 TFT_eSPI 跳过库默认配置

#define ST7789_DRIVER                // ST7789V 驱动
#define TFT_WIDTH  240
#define TFT_HEIGHT 240

#define TFT_MOSI   13                // D7  — GPIO13
#define TFT_SCLK   14                // D5  — GPIO14
#define TFT_CS     -1                // CS 接地
#define TFT_DC      0                // D3  — GPIO0
#define TFT_RST     2                // D4  — GPIO2
#define TFT_BL      5                // GPIO5 — 背光 (NPN 控制)
#define TFT_BACKLIGHT_ON LOW         // 低电平亮

#define SPI_FREQUENCY  20000000      // 20 MHz

// 仅加载代码实际使用的字体（节省 Flash 空间）
#define LOAD_FONT2                   // 小字体
#define LOAD_FONT7                   // 7 段数码管大字体
#define SMOOTH_FONT                  // 自定义 VLW 字体 (ZdyLwFont_20, FxLED_32)
```

## 上传

### 串口上传（首次烧录）

```bash
pio run -t upload
```

- 默认端口：`/dev/cu.usbserial-210`（macOS）/ `COMx`（Windows）
- 波特率：`115200`
- 如遇上传超时，手动按 **FLASH + RST** 进下载模式再试

### OTA 上传（设备联网后）

取消 `platformio.ini` 中 OTA 相关注释后：

```bash
pio run -t upload --upload-port 192.168.1.230
```

## 功能介绍

| 功能 | 说明 |
|------|------|
| 时钟显示 | NTP 校时，显示时:分:秒、星期、月/日 |
| 动画 | 5 种动画（乒乓 / 龙猫跳绳 / 龙猫转圈 / 太空人 / 跑步），通过 Web/MQTT 切换 |
| 天气 | 通过 `weather.com.cn` API 获取实时天气、温度、湿度、PM2.5 |
| Web 控制 | 内置 HTTP 服务器，可切换动画、设置城市代码、字体颜色、重启 |
| MQTT 控制 | 订阅 `esp8266/#`，远程切换动画和 LED |
| OTA 升级 | 支持 ArduinoOTA 无线升级 |

## 配网方式

### 方式一：EEPROM 存储的 WiFi（自动）

`connect_wifi()` 先从 EEPROM 读取上次保存的 WiFi SSID/密码，尝试连接。

### 方式二：SmartConfig 手机配网（首次 / 换网）

当 EEPROM 中的 WiFi 连不上（超过 120 次重试 ≈ 60 秒），自动进入 SmartConfig 模式。
手机连接 2.4G WiFi → 打开「ESP8266 SmartConfig」App 或安信可小程序 → 发送 WiFi 密码。
配网成功后 SSID/密码自动保存到 EEPROM，下次开机免配。

## 注意事项

> 代码审查记录详见 [`code-review.md`](./code-review.md)

- 开发板**无本地传感器**，温湿度数据来自网络天气 API
- SD 卡支持已禁用（`lib_ignore`），以节省 Flash 空间
- 动画帧数据通过查表替代 switch-case，大幅精简代码体积
